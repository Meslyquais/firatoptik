# Fırat Optik - PRD

## Problem Statement
Fırat Optik için siyah-beyaz, lüks moda dergisi estetiğinde (Vogue/Harper's Bazaar), modern ve elegant bir optik mağaza websitesi.

## Architecture
- **Frontend**: React + Tailwind CSS + Shadcn UI
- **Backend**: FastAPI + MongoDB
- **Auth**: JWT-based admin authentication
- **Fonts**: Playfair Display (headings) + Inter (body)

## User Personas
1. **Premium Müşteri**: Lüks marka gözlük arayan, moda bilinçli
2. **Optik Kalite Arayan**: Teknik cam kalitesine önem veren
3. **Admin/Mağaza Sahibi**: Ürün ve marka yönetimi yapan

## Core Requirements
- Hero slider (3 slayt: Moda, Teknik, Güven)
- Sonsuz marka logosu kaydırması (Zeiss, Essilor, Hoya vb.)
- 12 editorial marka bloğu (Dior, Gucci, Cartier, Tom Ford, Celine, Prada, Miu Miu, Bvlgari, Fendi, Bottega Veneta, Persol, Etnia Barcelona)
- Marka detay sayfaları + ürün galerisi
- Admin paneli (marka/ürün CRUD)
- İletişim (telefon + WhatsApp)
- Teknik mükemmellik bölümü

## What's Been Implemented (March 2025)
- Full-stack app: React frontend + FastAPI backend + MongoDB
- Hero slider with 3 Turkish-language slides and auto-rotation
- Infinite brand marquee (16 technical brands)
- 12 editorial brand blocks with B&W → color hover effect
- Brand detail pages with product gallery
- Admin login + dashboard with full CRUD for brands & products
- Contact footer with phone, WhatsApp, email, address
- Technical Excellence section
- Loading screen animation
- Responsive design
- All tests passing 100%
